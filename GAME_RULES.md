# Simulation Game — 인터랙션 규칙 정리

---

## 1. 캐릭터 이동

| 입력 | 동작 |
|------|------|
| `W` `A` `S` `D` | 캐릭터 이동 (카메라 yaw 기준 상대 방향) |

- 이동 시 캐릭터가 이동 방향을 바라봄
- 이동 중 walking bob 애니메이션
- 이동 영역 ±50 클램프
- **상호작용 메뉴/래디얼 메뉴 열린 상태**: WASD 이동 차단

---

## 2. 카메라 조작

| 입력 | 동작 |
|------|------|
| `←` `→` | 카메라 좌우 회전 |
| `↑` | 카메라 위로 |
| `↓` | 카메라 아래로 |
| **우클릭 드래그** | 카메라 회전 |
| **좌클릭 드래그** | 카메라 패닝 (캐릭터에서 벗어남) |
| **휠 위/아래** | 줌 인/아웃 (거리 3~20) |
| `+` / `-` | 줌 인/아웃 |

### 카메라 패닝 복귀
- WASD로 캐릭터 이동하면 카메라 자동 재포커스

### 카메라 차단
- **상호작용 메뉴 열림**: 방향키/WASD 카메라 회전 차단, 휠 줌 차단
- **래디얼 메뉴 열림**: 방향키 카메라 회전 차단

## 3. 타겟팅 시스템 (FOV 기반)

### 시야각 콘
- **캐릭터가 바라보는 방향** 기준 (카메라 방향 아님)
- 반각: **30°** | 사거리: **6m**
- 콘 원점: 캐릭터 뒤 5유닛 오프셋 (앞쪽 넓은 범위 확보)

### 두 개의 독립 큐

| 큐 | 키 | 동작 |
|----|-----|------|
| **사물 큐** | `F` | FOV 내 사물 스택. `V`로 전환 |
| **NPC 큐** | `E` | 가장 가까이+가운데 1명. 전환 불가 |

### 타겟 표시
| 종류 | 표시 |
|------|------|
| Primary | 키 아이콘(`F`/`E`, 컨트롤러: `A`/`X`) + 이름 (머리 위) |
| Secondary (사물만) | 흰색 원형 ring 20px + glimmer 반짝임 |

### 타겟 리스트 (화면 우측)
- **사물만** 표시 (NPC 제외)
- Active 항목: 왼쪽에 `▲[V]▼` 인디케이터 (absolute), 내부 오른쪽에 `[F]` 키
- 1개일 때 V 표시 숨김
- 메뉴 열리면 리스트 숨김

### 마우스 호버 & 클릭
- 오브젝트/NPC/플레이어 호버 시 emissive 하이라이트 + 커서 pointer
- **좌클릭** (드래그 5px 미만) → 상호작용 메뉴 열림
- 자기 캐릭터 클릭 → 셀프 상호작용 메뉴
- 빈 공간 클릭 → 열린 메뉴 닫힘
- 메뉴 열린 상태에서 다른 오브젝트 클릭 → 기존 닫고 새 메뉴 열기

## 4. 상호작용 메뉴 (2뎁스 시스템)

### 열기
| 입력 | 대상 |
|------|------|
| `F` | FOV 내 primary 사물 |
| `E` | FOV 내 primary NPC |
| `C` | 셀프 (자기 캐릭터). 다시 `C` 누르면 닫기 |
| **좌클릭** | 호버된 대상 |

### UI 레이아웃
- 화면 우측에 세로 pill 리스트 (반투명 blur)
- `›` chevron은 **2뎁스 있는 항목에만** 표시

### 2뎁스 사이드 바이 사이드
- 1뎁스 (좌): 유지. 2뎁스 진입 시 `focused` 해제 → `depth1-selected` (배경만, 외곽선/F미리보기 없음), 비포커스 항목 흐리게
- 2뎁스 (우): 새 패널로 오른쪽에 표시. 간격 ~10px

### 상단 키 가이드 (입력 방식별)
| 입력 방식 | 가이드 |
|----------|--------|
| 키보드 | `[Q] 뒤로/닫기` · `[↑↓] 탐색` |
| 마우스 | `[우클릭] 뒤로/닫기` |
| 컨트롤러 | `[B] 뒤로/닫기` · `[D-pad] 탐색` |

- 포커스된 아이템 왼쪽에 `[F]` (컨트롤러: `[A]`) 배지 직접 표시 (absolute, 가이드에는 미표시)

### 마우스 키 힌트 규칙
- **마우스 hover 시**: 키 힌트 숨김
- **휠 스크롤 시**: 휠 클릭 SVG 아이콘 표시

### 마우스 포커스 규칙
- 마우스로 열었을 때: **포커스 기본값 없음** (hover/방향키로 포커스)
- 키보드/컨트롤러로 열었을 때: **첫 항목 자동 포커스**

### 메뉴 내 조작
| 입력 | 1뎁스 | 2뎁스 |
|------|-------|-------|
| `↑`/`W` | 포커스 위 (**루프**) | 포커스 위 (**루프**) |
| `↓`/`S` | 포커스 아래 (**루프**) | 포커스 아래 (**루프**) |
| `V` | 아래방향 포커스 루프 | 아래방향 포커스 루프 |
| `→`/`D` | 2뎁스 진입 (서브메뉴 있을 때) | — |
| `←`/`A` | — (error 사운드) | 1뎁스로 뒤로 |
| `F`/`Enter` | 2뎁스 진입 또는 실행 | 실행 |
| `Q` | 메뉴 닫기 | 1뎁스로 **뒤로** (포커스 복원) |
| `ESC` | **즉시 닫기** | **즉시 닫기** |
| **우클릭** | 닫기 | 뒤로 |
| **휠 위/아래** | 포커스 이동 | 포커스 이동 |
| **휠 클릭** | 포커스된 항목 선택 | 포커스된 항목 선택 |

### 상호작용 메뉴 열린 상태 차단
- WASD 캐릭터 이동 차단
- 방향키 카메라 회전 차단
- 휠 줌 차단, 휠클릭 래디얼 차단

### NPC 메뉴 (1뎁스 → 2뎁스)
| 1뎁스 | 2뎁스 |
|-------|-------|
| 그룹 `›` | 파티 초대, 파티 떠나기 |
| 개인 행동 `›` | 춤추기, 앉기, 손흔들기 |
| 스마트폰 `›` | 전화하기, 문자 보내기, 사진 보여주기 |
| 상호 행동 `›` | 싸우기, 이름 묻기, 대화하기, 칭찬하기, 농담하기 |

### 셀프 상호작용 메뉴 (1뎁스 → 2뎁스)
| 1뎁스 | 2뎁스 |
|-------|-------|
| 그룹 `›` | 파티 만들기, 파티 해산 |
| 개인 행동 `›` | 춤추기, 앉기, 스트레칭, 손흔들기 |
| 스마트폰 `›` | 스마트폰 보기, 셀카 찍기, 음악 듣기 |
| 셀프 행동 `›` | 앉아서 신문읽기, 불러모으기, 명상하기, 하늘 보기 |

### 사물 메뉴 (1뎁스만, chevron 없음)
- 오브젝트별 고유 상호작용 직접 실행

## 5. 래디얼 메뉴 (원형 메뉴)

### 열기 (keyup 방식 — 조합키 충돌 방지)
| 입력 | 방식 |
|------|------|
| `Ctrl` / `X` | keyboard (release 시, 조합키면 무시) |
| **휠 클릭** (release, 드래그면 무시) | mouse |
| **Y** (컨트롤러) | controller |

- 같은 키 다시 → 닫기
- `Q` / `ESC` / 우클릭으로도 닫기
- 열리면 **HUD + 타겟 리스트 숨김**

### UI (Glass Ring, 560px)
- 투명 유리 링, **정확히 8등분** SVG
- 슬롯: **아이콘만** 표시 (SVG sector가 전체 조각 클릭/hover 처리)
- 포커스: 해당 조각 **하늘색 fill**

### 슬롯 키 힌트 (입력 방식별, SVG 마우스 아이콘)
| 상태 | 키보드 | 마우스 |
|------|--------|--------|
| 포커스 없음 | 각 슬롯에 `1`~`8` 표시 | 힌트 없음 |
| 포커스 있음 | 포커스 슬롯에 `F` 표시 | 포커스 슬롯에 좌클릭 SVG 아이콘 |

- 마우스 힌트는 SVG 아이콘: 좌클릭, 우클릭, 휠클릭, 휠스크롤

### 중앙 영역
| 상태 | 표시 |
|------|------|
| 포커스 없음 (keyboard) | "Radial Menu" + `1-8` |
| 포커스 없음 (mouse) | "Radial Menu" + 휠스크롤 SVG 아이콘 |
| 포커스 없음 (controller) | "Radial Menu" + `L-stick` |
| 포커스 있음 | **타이틀** + **설명** |

### 하단 키 가이드 (클릭 가능한 버튼)
| 키보드 | 마우스 | 컨트롤러 |
|--------|--------|---------|
| `[1-8]/[F]` 선택 | `[좌클릭]` 선택 | `[A]` 선택 |
| `[방향키]` 탐색 | `[휠스크롤]` 탐색 | `[L-stick]` 탐색 |
| `[Q]` 닫기 | `[우클릭]` 닫기 | `[B]` 닫기 |
| `[Tab]` 커스텀 | `[Tab]` 커스텀 | `[Start]` 커스텀 |

### 조작
| 입력 | 동작 |
|------|------|
| `1`~`8` | 즉시 실행 |
| **방향키** (조이스틱식 8방향) | 포커스. 대각선 지원 |
| `F`/`Enter` | 포커스된 슬롯 실행 |
| **좌클릭** (조각) | 해당 슬롯 **직접 실행** |
| **휠 스크롤** | 포커스 순환 (루프) |
| **휠 클릭** (열린 상태) | 포커스된 슬롯 실행 |
| `Q` / `ESC` / 우클릭 | 닫기 |
| `Tab` | 커스텀 모드 진입 |

### 래디얼 열린 상태 차단
- 방향키 카메라 회전 차단

## 6. 래디얼 메뉴 커스텀 모드

### 진입/나가기
- `Tab` (래디얼 열린 상태) 또는 하단 커스텀 버튼 클릭
- 컨트롤러: `Start` (래디얼 열린 상태)
- `Q` / `ESC` / `B`(컨트롤러) 로 나가기 (변경 있으면 확인 팝업)
- **게임 일시정지** (커스텀 모드 동안 게임 멈춤)

### UI
- dim 배경 + 중앙 래디얼 (동일 glass ring) + 우측 리스트 패널

### 강제 단계 순서 (키보드/컨트롤러 전용, 마우스는 자유)
- **Step 1**: 래디얼 슬롯 선택 — L-stick/D-pad/방향키로 래디얼 슬롯 탐색
- **Step 2**: 액션 선택 — 슬롯 선택 후, 같은 입력(L-stick/D-pad/방향키)이 액션 리스트 탐색으로 전환
- 래디얼 하단에 현재 단계 가이드 메시지 표시
- 포커스된 아이템에 키 힌트 `[F]`/`[A]` 표시

### L-stick 탐색
- **연속 탐색** (smooth): L-stick 기울임에 따라 부드럽게 탐색

### 래디얼 (중앙)
- 슬롯 hover → 중앙에 해당 슬롯 내용 표시
- 포커스 아웃 → 선택된 슬롯 정보 또는 empty state
- 액션 선택 시 → 모든 슬롯에 파란 glow "드롭 가능" 표시, 빈 슬롯 `+` 아이콘

### 중앙 버튼 (초기화 + 완료)
- **초기화 버튼**: 링 게이지 (마우스 클릭=즉시, 키보드/컨트롤러=`R` 길게 누르기)
- **완료 버튼**: 링 게이지 (마우스 클릭=즉시, 키보드/컨트롤러=`Space` 길게 누르기)
- 링 게이지: longpress 진행률 원형 표시

### 리스트 패널 (우측)
- 검색 (`F3` 포커스, 컨트롤러: `L3` — 입력 필드 안에 배지 표시), 고정 높이 420px 스크롤
- **카테고리 탭**: `Q`/`E` (키보드), `LB`/`RB` (컨트롤러)로 전환
- "없음" 항목으로 슬롯 비우기

### 배정 (양방향)
- 슬롯 → 리스트 또는 리스트 → 슬롯
- 슬롯 선택 + `Del` → 비우기

### 확인 팝업 (변경 있을 때 나가기)
- "저장 안 함" (`Enter`/`A`) → 스냅샷 복원 (변경 내용 버리기)
- "계속 편집" (`ESC`/`B`) → 팝업 닫기

## 7. HUD UI

### 구성
| 위치 | 내용 |
|------|------|
| 좌상단 | 위치명 + 빌드 버전 |
| 상단 중앙 | 툴바 (포토 모드 + 전체 메뉴만) |
| 좌측 중앙 | **컨트롤 가이드** (폰트 22px, kbd 진하게) |
| 좌하단 | 시간, 날씨, 게임 속도 |
| 하단 중앙 | 가방/스마트폰 버튼 + 초상화 + 욕구 게이지 |
| 우하단 | 통화 + 건축모드/캐릭터커스텀 버튼 |
| 우상단 | **(?) 가이드** 버튼 |
| 우측 | 홈/채팅 (시뮬레이션 외 기능 안내만) |

### HUD 배치 상세
| 영역 | 포함 항목 |
|------|----------|
| 상단 툴바 | 포토 모드 + 전체 메뉴 |
| 하단 욕구(mood) 영역 | 가방(`B`) + 스마트폰(`P`) |
| 우하단 단축키 영역 | 건축모드(`I`) + 캐릭터 커스텀 |

### 인게임 툴팁
- 버튼 hover 시 이름 + 단축키 표시 (위/아래 자동 위치 조정)
- 대상: 가방`B`, 스마트폰`P`, 건축모드`I`, 캐릭터 커스텀, 포토모드, 전체 메뉴`ESC`

### 컨트롤 가이드 (디바이스별 자동 전환)
| 키보드 | 컨트롤러 |
|--------|---------|
| `WASD` 이동 | `L-Stick` 이동 |
| `F` 사물 | `A` 사물 |
| `E` NPC | `X` NPC |
| `Ctrl`/`X` 래디얼 | `Y` 래디얼 |
| `V` 타겟 전환 | `LB` 타겟 전환 |
| `C` 셀프 | `L3` 셀프 |
| `U` HUD 숨기기 | — |
| `ESC` 메뉴 | `Start` 메뉴 |

### 단축키
| 키 | 동작 |
|----|------|
| `U` | HUD 숨기기 |
| `B` | 가방 |
| `P` | 스마트폰 |
| `I` | 건축모드 (전체화면, 토글) |
| `/` | 개발자 모드 |

### 전체화면 모드
| 모드 | 설명 |
|------|------|
| 건축모드 (`I`) | 전체화면, 카테고리+아이템 그리드, `← 뒤로` 버튼 |
| 캐릭터 커스텀 | 전체화면, 얼굴/헤어/상의/하의/신발/악세서리, `← 뒤로` 버튼 |
| 포토 모드 | **투명 배경** + 흰 사각 프레임 오버레이 (씬 보임), 하단 툴바, `← 뒤로가기` 버튼 |

### 패널 규칙
- phone, bag, build, charcustom, photo 중 하나만 활성 (열면 나머지 닫힘)
- 래디얼 메뉴의 건축/스마트폰/가방 슬롯 → 실제 패널 열기 연결
- `ESC` 우선순위: 확인팝업 → 커스텀 → 래디얼 → 상호작용 → 패널 → 게임메뉴

---

## 8. 게임 메뉴

- `ESC` / `Q` 또는 `Start`(컨트롤러)
- `Q`로도 닫기 가능
- 전체화면 dim + blur, 중앙 팝업
- 우클릭으로도 닫기
- 열린 동안 모든 키 차단
- 항목: 계속하기, 저장, 불러오기, 설정, 도움말, 타이틀로

---

## 9. 사운드 시스템 (Web Audio API)

| 사운드 | 용도 | 특성 |
|--------|------|------|
| `focus` | 포커스 이동 | 짧은 틱 (800→600Hz) |
| `error` | 불가능한 동작 | 낮은 버즈 (150Hz) |
| `execute` | 액션 실행 | 밝은 확인음 (600→1200Hz) |
| `open` | 메뉴 열기 | 상승 톤 (400→800Hz) |
| `close` | 메뉴 닫기 | 하강 톤 (600→300Hz) |

- 디버그 오버레이: 위치, yaw, FOV, 사물 큐, NPC 타겟

## 11. 입력 디바이스 자동 감지

- 키보드 입력 → `keyboard` / 마우스 클릭 → `mouse` / 게임패드 → `controller`
- 모든 가이드 UI가 자동 전환

### SVG 마우스 아이콘
- 힌트에 사용되는 SVG 아이콘 4종: **좌클릭**, **우클릭**, **휠클릭**, **휠스크롤**

### 컨트롤러 매핑
| 버튼 | 동작 |
|------|------|
| L-Stick | 이동 / 래디얼 슬롯 탐색 (연속, smooth) |
| R-Stick | 카메라 |
| A | F / 선택 |
| B | Q / 뒤로 / 닫기 (ESC와 동등) |
| X | E / NPC |
| Y | 래디얼 메뉴 토글 |
| LB | V / 타겟 전환 (커스텀 모드: 카테고리 탭 좌) |
| RB | 커스텀 모드: 카테고리 탭 우 |
| Start | ESC / 메뉴 (래디얼 열린 상태: 커스텀 모드 진입) |
| D-pad ↑↓ | 메뉴 탐색 (**래디얼 슬롯 탐색에는 사용 안 함 — L-stick 전용**) |
| L3 | C / 셀프 상호작용 (커스텀 모드: 검색 포커스) |

## 12. UI State 다이어그램

```
[기본 상태]
  ├── Ctrl/X (release) / 휠클릭(release) / Y ──→ [래디얼 메뉴]
  │     ├── 같은키/Q/ESC/우클릭 ──→ [기본]
  │     ├── Tab / 커스텀 버튼 / Start(컨트롤러) ──→ [커스텀 모드] (게임 일시정지)
  │     │     ├── Space 길게 / 완료 버튼 클릭 ──→ [기본] (저장)
  │     │     ├── R 길게 / 초기화 버튼 클릭 ──→ 슬롯 초기화
  │     │     ├── Q/ESC/B (변경無) ──→ [기본]
  │     │     └── Q/ESC/B (변경有) ──→ [확인 팝업]
  │     │           ├── Enter/A ──→ [기본] (변경 버리기)
  │     │           └── ESC/B ──→ [커스텀 모드] (계속 편집)
  │     ├── 좌클릭/숫자/F/Enter/휠클릭 ──→ [기본] (실행)
  │     └── 방향키 ──→ 8방향 포커스 → F/Enter/휠클릭 ──→ [기본] (실행)
  │
  ├── F/E/C/좌클릭 ──→ [상호작용 1뎁스]
  │     ├── →/D/F/Enter ──→ [상호작용 2뎁스]
  │     │     ├── F/Enter/휠클릭 ──→ [기본] (실행)
  │     │     ├── ←/A/Q/우클릭 ──→ [1뎁스] (뒤로)
  │     │     └── ESC ──→ [기본]
  │     ├── Q/우클릭/빈곳클릭 ──→ [기본]
  │     ├── C (셀프 열려있을 때) ──→ [기본]
  │     ├── 다른 오브젝트 클릭 ──→ [기본] → [새 1뎁스]
  │     └── ESC ──→ [기본]
  │
  ├── B ──→ [가방] ──→ B/ESC ──→ [기본]
  ├── P ──→ [스마트폰] ──→ P/ESC ──→ [기본]
  ├── I ──→ [건축모드 전체화면] ──→ I/ESC/←뒤로 ──→ [기본]
  ├── 캐릭터커스텀 ──→ [전체화면] ──→ ESC/←뒤로 ──→ [기본]
  ├── 포토모드 ──→ [투명+흰 프레임 오버레이] ──→ ESC/←뒤로 ──→ [기본]
  ├── ESC/Q ──→ [게임 메뉴] ──→ ESC/Q/우클릭 ──→ [기본]
  │
  ├── U ──→ HUD 숨기기
  ├── V ──→ 사물 타겟 전환
  └── / ──→ 개발자 모드
```
